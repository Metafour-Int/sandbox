#Git HELP
##cvs update?
Git doesn't have the update command so, to pull updated from a branch, you do ``git pull origin <branch>``

##merge branch
``git checkout <target_branch>``

``git merge <working_branch>``

##tagging
``git tag -a x.x.x -m 'message'```

``git push --tags``
